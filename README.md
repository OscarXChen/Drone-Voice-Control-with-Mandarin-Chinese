# Drone-Voice-Control-with-Mandarin-Chinese
A project that using mandarin Chinese to control drones

这是一个致力于使用普通话的自然语音来控制无人机的本科毕设项目。

这是一个勉强可行的方案，项目事实上没有完全达到预期目标，但我认为尤其对本科生而言，我已完成的部分仍然有较好的参考价值。

将项目记录在此，希望可以帮到你。
